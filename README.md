# Raksha-AI
AI based emergency responce system
**"Instant Care. In Every Indian Language."**

👩‍💻 Team: Tech Titans
🏫 Institution: Chennai Institute of Technology  

---

🧩 Problem Statement – The “Golden Hour” Gap

- 🌍 **Language Barriers**  
  Emergency systems mostly support only English/Hindi, making it hard for regional users.

- ⏱ **Loss of Golden Hour**  
  Delay in identifying symptoms reduces survival chances.

- 📞 **Delay in Contacting Help**  
  People don’t know which hospital to contact or how to share location.

- 👴 **Accessibility Issues**  
  Elderly and visually impaired people struggle to seek help.

---

💡 Our Solution – RakshaAI

- 🗣 **Multilingual Voice Interface**  
  Supports regional languages for hands-free emergency help.

- 🚑 **Instant Emergency Connect**  
  Auto-calls nearest hospitals and shares live GPS location.

- 🧠 **AI-Powered First Aid**  
  Provides step-by-step voice instructions (CPR, etc.)

- ⚠️ **Smart Distress Detection**  
  Detects emergency severity using voice tone and keywords.

---

⚙️ How It Works

1. 🎤 User speaks in regional language  
2. 🌍 Speech converted using Bhashini API  
3. 🧠 AI analyzes emergency using NLP  
4. 🚑 Alerts hospital + shares GPS  
5. 🗣 Provides voice guidance until help arrives

- 🩺 Saves lives during Golden Hour  
- 📍 Real-time location sharing
